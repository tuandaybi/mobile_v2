# iOS Prober UI (Tkinter)

Ứng dụng desktop để:
- Liệt kê iPhone/iPad cắm USB
- Đọc thông tin (UDID, Serial, iOS, pin, dung lượng…)
- Gửi JSON lên Laravel API (Bearer Sanctum)

## Cài đặt
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
```

## Chạy
```bash
python main_gui.py
```
